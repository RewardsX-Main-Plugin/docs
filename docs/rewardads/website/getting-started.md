# Getting Started

Transform your applications into revenue-generating platforms with RewardsX. Our SDK enables seamless ad integration and reward systems that engage users while maximizing your earnings.

## Step 1: Create Your Platform

Navigate to [dash.rewardsx.net](https://dash.rewardsx.net) and register your first platform in under 2 minutes. The dashboard provides comprehensive tools for managing your monetization strategy:

- **Real-time Analytics** - Track performance metrics and user engagement
- **Revenue Management** - Monitor earnings and configure payout settings
- **SDK Integration** - Access detailed implementation guides and code samples

## Step 2: Complete Platform Verification

> **⚠️ Important:** Platform verification is mandatory before accessing SDK features or generating revenue.

This security process is straightforward, just plug the SDK into your platform and get verified!

## Step 3: Integrate & Launch

Once verified, download the SDK and follow our [integration guides](/rewardads/integration) to implement ads in your application. Most developers complete integration within a few minutes.

### Popular Integration Options:

- **Web SDK** - For web applications and games
- **Minecraft Plugin** - Dedicated to Minecraft servers
