# Getting Started

DiscordRewards is an addon for RewardsX that enables players to earn in-game rewards by sending a set number of messages on your Discord server.

## Overview
DiscordRewards connects Discord message activity with RewardsX rewards on your Minecraft server. Players send messages in configured channels, and when they reach a configured message threshold, RewardsX automatically grants a reward.

## Requirements
- A Minecraft server with the [RewardsX](https://spi.rewardsx.net) plugin installed
- [DiscordLib](https://cdn.rewardsx.net/downloads/spigot/discordlib/DiscordLib-1.0.0.jar) plugin Installed. (required for DiscordRewards to work correctly)
- A Discord server where your community chats
- A Discord bot created in the [Discord Developer Portal](https://discord.com/developers/applications), with its bot token available
- Basic ability to edit YAML configuration files on your server
